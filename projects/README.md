# Projects — Chunking & RAG Learning Track

Build these projects in order. Each one adds one new layer on top of the previous.
By the end you will have built every component of the capstone yourself, from scratch.

---

## The Progression

```
P1 — Chunking Playground
     "Understand how documents are cut"
          │
P2 — My First RAG
     "Ask questions, get answers from your own documents"
          │
P3 — PDF RAG
     "Work with real PDF files"
          │
P4 — Strategy Battle
     "Compare chunking strategies side by side"
          │
P5 — Complex Document RAG
     "Handle tables, images, and scanned PDFs"
          │
P6 — Production RAG
     "Add reranking, hybrid search, and evaluation"
          │
P7 — University Document Assistant (Capstone)
     "Full production system for university employees"
```
